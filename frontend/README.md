# 选课平台前端

## 📋 概述

这是一个现代化的学生选课前端应用，支持多服务架构（8080/8081/8082），具备自动降级功能。

## 🏗️ 架构说明

前端调用三个后端微服务：

| 服务 | 端口 | 用途 | 优先级 |
|------|------|------|--------|
| **basic-service** | 8080 | 用户认证、课程管理、学生管理 | 基础服务 |
| **selection-service** | 8081 | 选课/退课操作（高并发优化） | ⭐ 优先 |
| **statistics-service** | 8082 | 实时排行榜、统计数据 | ⭐ 优先 |

### 智能降级机制

- **选课/退课**: 优先使用 8081，失败时自动降级到 8080
- **实时统计**: 优先使用 8082，失败时自动降级到 8080
- **基础数据**: 始终使用 8080

## 📁 主要文件

- `index.html` — 页面入口，包含环境配置
- `styles.css` — 样式文件
- `app.js` — 前端逻辑，调用 API
- `ENV_CONFIG.md` — 详细的环境配置文档
- `switch-env.bat` — Windows 环境切换工具

## 🚀 快速启动

### 方式一：使用 Python HTTP 服务器（推荐）

```bash
# 在 frontend 目录打开终端
python -m http.server 5500

# 打开浏览器访问
# http://localhost:5500/
```

### 方式二：直接打开 HTML 文件

直接在浏览器中打开 `index.html` 文件（部分浏览器可能限制本地文件的 AJAX 请求）

### 方式三：使用 VS Code Live Server

安装 Live Server 扩展后，右键 `index.html` 选择 "Open with Live Server"

## 💡 使用说明

### 登录

1. 输入用户名和密码
2. 点击“登录”按钮
3. 系统会自动识别角色（学生/管理员）

### 课程列表页

- **学生角色**：可以查看课程信息，进行选课/退课操作
- **管理员角色**：可以输入学号，代学生选课/退课

### 统计页

- **库存充足榜 Top10**：显示剩余名额最多的课程（来自 8082）
- **累计选课次数**：显示系统总选课次数（来自 8082）

### 管理面板（仅管理员）

- **课程管理**：添加/删除课程
- **学生管理**：添加/删除学生，查看学生列表

## ⚙️ 技术特性

### 1. 环境变量配置

通过 `window.BACKEND_HOST` 变量轻松切换开发和生产环境。

### 2. 自动降级

当优先服务不可用时，自动降级到基础服务，保证核心功能可用。

### 3. 响应式 UI

- 表格化展示数据
- 清晰的视觉反馈
- 角色权限控制

### 4. 错误处理

- 友好的错误提示
- 控制台警告信息
- 降级日志记录

## ⚠️ 注意事项

### 1. 后端服务要求

确保以下服务已启动并配置了 CORS：

```bash
# basic-service (必须)
mvn spring-boot:run -pl basic-service

# selection-service (推荐)
mvn spring-boot:run -pl selection-service

# statistics-service (推荐)
mvn spring-boot:run -pl statistics-service
```

### 2. CORS 配置

如果前端和后端不在同一域名/端口，需要在后端配置 CORS。参考 `ENV_CONFIG.md`。

### 3. 浏览器兼容性

建议使用现代浏览器：
- Chrome 90+
- Firefox 88+
- Edge 90+
- Safari 14+

### 4. 调试技巧

打开浏览器控制台（F12）：
- **Network 标签**：查看 API 请求和响应
- **Console 标签**：查看降级警告和错误信息

## 🐛 故障排查

### 问题：无法连接到后端

**解决方案**：
1. 检查后端服务是否启动
2. 确认端口是否正确（8080/8081/8082）
3. 检查 `window.BACKEND_HOST` 配置
4. 查看浏览器控制台的错误信息

### 问题：选课时提示失败

**解决方案**：
1. 检查是否已登录
2. 确认学号是否正确
3. 查看控制台是否有降级提示
4. 检查课程容量是否已满

### 问题：统计数据不显示

**解决方案**：
1. 检查 statistics-service (8082) 是否启动
2. 查看控制台是否有降级提示
3. 确认数据库中是否有选课记录
